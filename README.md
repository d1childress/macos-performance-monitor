# macOS Performance Monitor

A comprehensive system performance monitoring tool for macOS that tracks CPU, memory, disk, network, and process information.

## Features

- Real-time CPU monitoring (overall and per-core usage)
- Memory usage tracking
- Disk I/O statistics
- Network I/O monitoring
- Process information and resource usage
- JSON output support

## Requirements

```bash
pip install -r requirements.txt
```

## Usage

```bash
python3 macos_performance_monitor.py
```

## Dependencies

- psutil
- matplotlib (optional, for visualization)
- numpy (optional, for data analysis)

## Platform

macOS only
